## Как читать комментарии

Все замечания помечены грепаемым маркером `REVIEW[уровень]`. Найти все разом:

```bash
grep -rn "REVIEW\[" src/ pom.xml .gitignore
```

Уровни:

| Маркер | Что значит | Приоритет |
|---|---|---|
| `REVIEW[BLOCKER]` | Критично (безопасность / сломанный build). Чинить в первую очередь. | 🔴 |
| `REVIEW[BUG]` | Настоящий баг — код работает не так, как ожидается. | 🟠 |
| `REVIEW[DEBT]` | Долг с прошлых ревью (1.1-1.3), ещё не закрыт. | 🟡 |
| `REVIEW[THINK]` | Открытый вопрос. Ответь сам — это часть обучения, не просто фикс. | 🔵 |
| `REVIEW[NIT]` | Мелочь / стиль. Желательно, но не срочно. | ⚪ |
| `REVIEW[GOOD]` | Сделано хорошо — чтобы ты видел, что работает, а не только ошибки. | 🟢 |
